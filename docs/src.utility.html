src.utility.import_export.load_bootflow(callback)[source]

Prompt the user to choose a file and load a BootFlow instance from that file.

The file must be a JSON file containing BootFlow data. The loaded BootFlow instance is passed to the provided callback function.

Args:

callback (function): A function to be called with the loaded BootFlow instance.

src.utility.import_export.load_bootflow_from_directory(directory_path)[source]

Load a BootFlow instance from a JSON file located in the specified directory.

The file is expected to be named ‘bootflow.json’. If the file is found and successfully read, a BootFlow instance is created from the data. Otherwise, a message is printed, and None is returned.

Parameters:

directory_path (str): The path to the directory containing the ‘bootflow.json’ file.

Returns:

BootFlow or None: A BootFlow instance if the file was found and loaded; otherwise, None.

src.utility.import_export.new_boot_flow(old_bootflow, callback)[source]

Save the current BootFlow instance and create a new BootFlow instance.

The current BootFlow instance is saved to the default file path, and a new BootFlow instance is created and passed to the provided callback function.

Parameters:

old_bootflow (BootFlow): The current BootFlow instance to be saved. callback (function): A function to be called with the new BootFlow instance.

src.utility.import_export.save_as_bootflow(bootflow: BootFlow)[source]

Prompt the user to choose a file path and save the BootFlow instance to that file.

The default_file_path is updated to the new file path chosen by the user.

Args:

bootflow (BootFlow): The BootFlow instance to be saved.

src.utility.import_export.save_bootflow(bootflow: BootFlow)[source]

Save the given BootFlow instance to a JSON file.

If default_file_path is not set, the user is prompted to choose a file path. If default_file_path is set, the BootFlow is saved to that path.

Args:

bootflow (BootFlow): The BootFlow instance to be saved.
